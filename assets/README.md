# Logo θέση

Βάλε εδώ το αρχείο λογοτύπου με όνομα `logo.png`.

Το site το εμφανίζει αυτόματα στο header (στη θέση του κειμένου "GZ Engineering") από το path:

- `assets/logo.png`

## Background φυσικού αερίου

Η σελίδα `fysiko-aerio.html` χρησιμοποιεί ως hero background απευθείας URL από το Unsplash CDN μέσα στο `styles.css`.

Πηγή αρχικής φωτογραφίας: Unsplash, Malcolm Choong, "a pressure gauge attached to a pipe in a room":
https://unsplash.com/photos/a-pressure-gauge-attached-to-a-pipe-in-a-room-SLSly1r8wiM

Άδεια χρήσης: Unsplash License, δωρεάν χρήση για εμπορικούς και μη εμπορικούς σκοπούς χωρίς υποχρεωτική αναφορά δημιουργού.
